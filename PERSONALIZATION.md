# Personalization Guide

这份仓库现在已经是你的个人 IP 版本，不再只是原始范例。

## 你已经固定住的部分

- 头像：`assets/avatar.jpg`
- 主气质：日系清新插画、蜡笔感、贴纸描边、黄色背景
- 主色：蓝 / 黄 / 红三色
- 关键目标：看起来像你本人，而不是通用 AI 模板

## 先保留的东西

- `SKILL.md` 的 7 步工作流
- `references/layouts.md` 里的布局库
- `references/components.md` 里的组件库
- `assets/template-*.html` 的模板骨架

这些结构本身很好，不需要推倒重来。真正要换的是审美决策。

## 现在应该改的判断

### 1. 默认头像

所有出现头像的地方，默认都用 `assets/avatar.jpg`。

如果要做变体，只改：
- 表情
- 视角
- 裁切
- 场景

不要改：
- 眼镜
- 发型轮廓
- 头身比例
- 颜色骨架

### 2. 默认气质

把页面从“品牌展示”改成“个人创作者感”：

- 更像 sketchbook，而不是 corporate deck
- 更像知识博主，而不是公司官网
- 更像手写注释和图形记号，而不是标准化 UI

### 3. 默认节奏

- 背景偏暖
- 黄色负责高亮
- 蓝色负责主线条和主视觉
- 红色只做点睛
- 页面要有手绘留白，不要每一屏都塞满

## 继续做页面时的判断顺序

1. 先想这页是不是你的个人 IP 页面
2. 再想要不要用头像做首屏锚点
3. 再定是“知识感”还是“轻松感”
4. 最后才选布局和组件

## 快速替换规则

如果你要把别的页面也收进这套系统，优先做这几件事：

- 把旧头像换成你的头像
- 把蓝/黄/红的比例固定住
- 把标题气质统一到“温和、清晰、有内容”
- 把所有太商业化、太冷感、太通用模板的元素删掉

## 一句话版

这套系统的核心不是 Finn 的风格，而是：**有个人 IP 的手绘知识感**。
