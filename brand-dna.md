# Brand DNA - 个人 IP 基因

> 这是一套给 AI 看的个人品牌设计系统。所有页面、所有场景都要服从同一套审美判断。

---

## 三色系统

| 色名 | 色值 | 对应 | 用途 |
|------|------|------|------|
| 蓝 | `#2F6FDB` | 头发 / 主线条 | 主色调、标题、链接、关键标记 |
| 黄 | `#F4D758` | 背景 / 高亮 | 强调、装饰、badge、连接线、选中状态 |
| 红 | `#E86A5A` | 点缀 | CTA、标签、少量强调，不做主色 |

比例原则：蓝 60% · 黄 30% · 红 10%。

---

## IP 形象规范

固定视觉形象是 `assets/avatar.jpg`，以下元素尽量不要改：

- 戴细边圆框眼镜
- 蓝黑短发
- 温和安静的表情
- 米白衬衫 + 蓝色针织衫
- 黄色圆形背景
- 白色贴纸描边
- 蜡笔 / 彩铅手绘质感

### 使用规则

- `assets/avatar.jpg` 是默认头像
- 需要头像时优先用它
- 生成变体时只改表情、姿势、裁切，不改脸型、发型轮廓和整体色调
- 不要把这个 IP 画成女性化、商务精英感过强、或过度写实的头像

## 字体基因

### 核心原则

- 标题用衬线，正文用无衬线
- 英文做装饰，中文承载主要信息
- 版面要有“手绘但有秩序”的节奏

### 推荐字体池

| 场景 | 推荐 | 备注 |
|------|------|------|
| 英文装饰 / 标题 | `Fraunces` | 只作为轻量强调 |
| 英文手写 / 注释 | `Caveat` | 标注、批注、手绘感 |
| 中文标题首选 | `Noto Serif SC` | 稳定、可读、带一点知识感 |
| 中文正文 | `Noto Sans SC` + 系统栈 | 跨平台无衬线 |
| 代码 / 终端 | `Fira Code` | 技术内容专用 |

---

## 气质关键词

- 日系清新插画感
- 手绘蜡笔感
- 温和、聪明、亲和
- 像个人创作者，不像公司模板
- 有内容，但不压人
- 轻知识、轻表达、重识别度

---

## 视觉扩展原则

- 背景优先用暖白、米白、淡黄
- 文字不要纯黑，尽量用墨色
- 少量深色面板可以，但不要满屏压暗
- 不要把蓝紫渐变、neon、glassmorphism 当默认解法
- 不要把页面做成泛化的 SaaS 模板感

---

## 通用禁忌

| 类型 | 禁止 |
|------|------|
| 配色 | 蓝紫渐变、霓虹光效、过冷灰蓝、纯黑纯白大面积对撞 |
| 字体 | Inter / Roboto / Arial 作为默认气质 |
| 布局 | 所有 section 都居中、重复卡片网格、没有变化的纵向堆叠 |
| 动效 | bounce、elastic、无意义无限循环 |
| 图片 | stock photo 风、AI 味太重的摄影图 |
| 默认样式 | 浏览器原生 blockquote / ul / ol / table |

---

## 间距原则

- Section 间距：`clamp(80px, 12vh, 160px)`
- 内容块间距：`clamp(40px, 6vw, 100px)`
- 卡片内边距：`clamp(24px, 3vw, 44px)`
- 内容宽度：`max-width: 1300px`

---

## 自检问题

1. 一眼看上去是不是有自己的 IP，而不是通用模板？
2. 头像、颜色、字体、留白是不是在同一个气质里？
3. 截图发出去，会不会让人觉得像“你本人”？
